# Mr.AudIT - Audit Dashboard

## Description
This is a web-based IT audit dashboard created using HTML, CSS, and JavaScript. It shows the audit results of a supermarket.

## Technologies Used
- HTML
- CSS
- JavaScript

## Audit Areas
- Billing system
- Data security
- Network
- Backup system
- CCTV

## How to Run
Open index.html in your browser
